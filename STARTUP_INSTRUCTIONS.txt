═══════════════════════════════════════════════════════════════════
 ❗ IMPORTANT NOTES
═══════════════════════════════════════════════════════════════════

   ⚠️  MUST run as Administrator for network access!
       (Right-click → "Run as Administrator")

   ⚠️  Both devices must be on the SAME Wi-Fi network
       (Same router, same network name)

   ⚠️  Use the Network IP, NOT localhost
       ✅ Good: http://192.168.1.105:5001
       ❌ Bad:  http://localhost:5001 (only works on same PC)

   ⚠️  Disable VPN if connection fails
       (VPNs can block local network access)

 🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

   Can't connect from phone?
   ► Read: NETWORK_TROUBLESHOOTING.md (complete guide)
   ► Check: Both devices on same Wi-Fi
   ► Verify: Ran as Administrator
   ► Test: Can you access localhost:5001 on server PC?

   Server won't start?
   ► Check: Python installed (python --version)
   ► Run: pip install -r requirements.txt
   ► Check: Port 5001 not in use (netstat -an | findstr :5001)

   Upload not working?
   ► Check: Browser console for errors (F12)
   ► Look at: Terminal/PowerShell window for debug logs
   ► Verify: Files appear in uploads/ folder

 ✅ VERIFICATION CHECKLIST
