 🚀 HOW TO USE
═══════════════════════════════════════════════════════════════════

Step 1: Start the Server
────────────────────────────────────────────────────
cd C:\Users\v-hbonthada\WorkSpace\FileShare
python app.py

Step 2: Note the Network IP
────────────────────────────────────────────────────
Look for this in the output:

📍 SERVER ADDRESSES:
   • Local access:   http://localhost:5001
   • Network access: http://192.168.1.XXX:5001  ← USE THIS!

Step 3: Access from Mobile
────────────────────────────────────────────────────
On your phone/tablet:
1. Connect to the same Wi-Fi network
2. Open any web browser
3. Go to: http://192.168.1.XXX:5001
4. Enjoy the mobile-optimized interface!

Alternative: Use QR Code
────────────────────────────────────────────────────
1. On computer, go to "Network" tab
2. Scan the QR code with your phone
3. Browser opens automatically!
